# Task 2 – Mini CRM Application  
**Track Code:** FS (Full Stack Web Development)  
**Internship:** Future Interns  

---

## Project Overview

This project is a **Mini CRM (Customer Relationship Management) Web Application** developed as **Task 2** for the **Future Interns Full Stack Web Development Internship**.

The application allows users to manage customer leads, update lead status, track follow-ups, and view real-time analytics through a dashboard.

---

## Tech Stack

### Frontend
- React.js (Vite)
- HTML5
- CSS3
- Axios

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose

---

## Features

### Lead Management
- Add new leads (Name, Email, Status, Notes, Follow-up Date)
- Update lead status (New, Contacted, Converted)
- Edit notes and follow-up dates
- Delete leads

### Dashboard Analytics
- Total Leads count
- New Leads count
- Contacted Leads count
- Converted Leads count
- Visual analytics using:
  - Status cards
  - Progress bars
  - Donut chart with percentage distribution

### User Interface
- Sidebar navigation (Overview & Leads Dashboard)
- Clean and modern UI
- Responsive layout
- Real-time data updates
